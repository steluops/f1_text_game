# This is a prediction simulator using drivers' overall ratings, and car ratings determining possible outcomes of Formula 1 races
  on some F1 circuits.

## Driver and car ratings are already stored within the program. The user first chooses which of the five circuits to race at
  (more on the way).

## The circuits each have their own distinctive amount of laps which starts at 1, and incremented until the final lap.

## Once the lap number reaches its maximum, the program displays the drivers and cars from positions 1-22.
